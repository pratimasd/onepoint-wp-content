Custom Gutenberg blocks: carousels, hero banner, contact form, header and footer.

== Description ==

This plugin registers a set of custom Gutenberg blocks for use with block themes or classic themes. Blocks are built with WordPress scripts only (no custom webpack). Each block can be edited in the block editor and rendered on the front end via PHP.

Included blocks include image carousel, hero banner, technology carousel, client stories carousel, purpose cards carousel, contact form, site header, and site footer.

== Installation ==

1. Upload the plugin folder to `/wp-content/plugins/`.
2. Activate the plugin in the WordPress admin under Plugins.
3. Run `npm install` and `npm run build` in the plugin directory if you need to rebuild block assets.
